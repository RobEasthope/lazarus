Louise + Daniel


Client: Louise & Daniel Holly

Demo site

The back story

Louise + Daniel was a simple responsive microsite to give guests details about the wedding of Louise & Daniel Holly and allow them to RSVP without all the hassle of written invitations.

Technical notes

Originally built off using the Jekyll based Octopress framework using Bootstrap 3.0 for frontend support.

THe site has now been refactored into a simple static site using Jade templating, SCSS, Foundation for Sites, and Gulp
