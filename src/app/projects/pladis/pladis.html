The back story

The Pladis project was a brand new responsive, multi-lingual website for the Turkish Pladis holding group bringing together information on their Ulker, Mcvities, and Godiva holdings.

The CMS

Pladis was designed from the start as an international, multi-lingual site, with a requirement to be easily updated by communications staff without the need for specalist developer knowledge.

Using the fantastic Advanced Custom Fields (ACF) and WPML (WordPress Multi-Lingual) plugins we were able to create a fast, dymamic content management process that allows all aspects of the sites content to be created, edited, and rearranged by normal users.

Fullstack development

WordPress sites can sometimes get a bad name due to poor development and server practices rather than a core problem with WordPress itself. When initially approached by Bisqit, Pladis were hesitant to use WordPress due to security and reputational concerns. This was complicated by Bisqit not having full-time developers on staff to maintain their sites.

This created a requirement for a solid and secure tech stack to address Pladis's security concerns while still maintaining an easy-to-use, well documented development and deployment process that allowed new developers to get up and running quickly.

Both of these issues were solved by using the excellent open-source, Ansible based Tellis/Bedrock/Sage stack from roots.io. Built using sound development practices and extensive use of secure encryption to protect vital password keys, the entire site can set up on a new server with as little as three lines of terminal code.

Technical details

  • Wordpress CMS
  • Modern tech Ansible/Vagrant based tech stack via Roots
  • Custom CMS elements via Advanced Custom Fields
  • Multilingual support via WPML
  • Fully responsive
  • Package management via Composer and Bower
  • Front end code compilation via Gulp
